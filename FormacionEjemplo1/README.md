FormacionEjemplo1
==============
Ejemplo básico con Vaadin de un TextField, un Button y una Label.

Workflow
========
Para compilar lanzar en el cmd del proyecto "mvn install".

Para correr la aplicación, lanzar "mvn jetty:run" y abrir http://localhost:8080/ .
